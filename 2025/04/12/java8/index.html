Java SE(8)-泛型

概述

泛型（Generics）是Java提供的一种机制，让类、接口、方法在定义时可以指定类型参数，从而实现类型安全和代码复用。

泛型的继承

Java 泛型的类型是严格的类型检查，并不允许不同类型的泛型直接相互赋值。例如，ArrayList<Object>和ArrayList<String>是两种不同的泛型类型，即使String是Object的子类。

解决方法：

  1. 使用通配符(Wildcard):

上界通配符：List<? extends T> list，list是某种T的子类的列表。可以读，不能写，因为编译器不知道list是哪个子类。读取安全但是写入不安全

下界通配符：List<? super T> list，list是T的某个父类的列表。读出只能被视为为Object（因为不知道具体是哪个父类），可以写入T或其子类。写入安全但是读取受限

符合向上转型原则。
